## 📖 About

NeoLearn AI is a full-stack AI-powered learning platform designed to help students learn technical subjects efficiently using Artificial Intelligence.

Instead of switching between multiple websites, NeoLearn AI provides everything in one platform including:

- 🤖 AI Tutor
- 💬 AI Chat Assistant
- 📄 ATS Resume Analyzer
- 📚 PDF Learning Assistant
- 📝 AI Quiz Generator
- 📊 Learning Progress Tracker
- 💼 Internship Finder
- 🗺️ Personalized Learning Roadmaps

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YashDeshmukh495/NeoLearn-AI.git
```

Move into project

```bash
cd NeoLearn-AI
```

---

## Backend

```bash
cd backend

npm install

npm start
```

---

## Frontend

Simply open

```
frontend/index.html
```

or use Live Server in VS Code.

---

# 🔑 Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=5000

MONGO_URI=your_mongodb_uri

JWT_SECRET=your_secret_key

GEMINI_API_KEY=your_api_key
```
